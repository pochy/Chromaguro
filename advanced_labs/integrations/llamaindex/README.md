# LlamaIndex + Chroma Lab

この lab は任意です。

```bash
pip install -r requirements-integrations.txt
python advanced_labs/integrations/llamaindex/01_llamaindex_chroma.py
```

## 学ぶこと

```text
LlamaIndex は documents / nodes / index / query engine の抽象化を持つ。
Chroma は vector store として使える。
chunking と metadata の責任が LlamaIndex 側にも移るため、Chroma collection 設計を忘れない。
```
